## 🏥 AI-Adjusted Nursing Shortage Estimation Model

**Live App:** [https://drfarooqai.github.io/nurse.html](https://drfarooqai.github.io/nurse.html)

![AI Nursing Shortage Model](https://raw.githubusercontent.com/DrFarooqAi/DrFarooqAi.github.io/main/docs/nurse-preview.png)

An interactive quantitative simulator modelling how AI documentation automation mitigates the projected **5.8M global nursing deficit by 2030**.

**Features:**
- Dynamic PA-FTE formula: `PA-FTE = Baseline FTE × (1 + Adoption × Effectiveness × 0.85)`
- Real-time sliders for AI Adoption Rate & Effectiveness
- Regional breakdown (Global · Africa · Asia · Americas · Europe · Mediterranean)
- Live bar chart — Baseline vs. AI Gain vs. Net Shortage
- Scenario classification: Conservative / Moderate / Optimistic

**Data Sources:** WHO · BLS · OECD
